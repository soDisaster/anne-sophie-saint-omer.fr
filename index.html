Projets

Projet individuel

Ce projet consiste à réaliser une application permettant à l'utilisateur de créer pour un sujet donné une base de données sous-jacente. L'utilisateur pourra ajouter des entrées à cette base de données et en modifier le contenu. La structure de la base de données est dynamique. Cette application permet de faciliter le travail de catégorisation de l'utilisateur tout au long de son travail sans que celui-ci n'ait à déterminer une structure définitive. Ce projet repose sur HTML, CSS, PHP, jQuery et SQLite

Projet Javascript

Agar.io est un jeu d'action massivement multijoueur gratuit développé par Matheus Valadares. Le joueur contrôle une cellule, dont le but est de grossir le plus possible en absorbant des cellules plus petites sans être absorbée par des cellules plus grosses. Thomas et moi-même avons réalisé une version revisitée et simplifiée du jeu Agar.io. Elle repose sur du Javascript Objet, le framework PaperJS pour la manipulation du canvas et le framework Cobra pour les échanges de messages entre joueurs.

Ma première application Android

Le but de cette application est de gérer ses plantes d'intérieur. La couleur verte indique que la plante n'a pas besoin d'être arrosé. La couleur orange indique qu'il reste une journée ou que c'est le dernier jour pour arroser la plante. La couleur rouge indique qu'on a dépassé le délai pour arroser la plante. On peut modifier ou supprimer une plante en cliquant sur celle-ci. Si on clique longuement, on l'arrose. La date d'arrosage et la couleur sont modifiés.
